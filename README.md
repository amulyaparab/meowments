# meowMents

A social media app to capture all those adorable cat meowments! - https://meowments.vercel.app/

# Demo video

# Features
* User Feed
  * List all user posts
  * Sort by Date & Trending
* Post Actions
  * Create a Post
  * Like a post
  * Edit & Delete Post
  * Comment on Post
* User Profile
  * Add Avatars
  * Add Bio
  * Add Portfolio URL
  * Edit Profile
* Follow/Unfollow
* Explore Feed
* Bookmark Post
* Authentication
  * Sign-up
  * Login
  * Logout
* Search Users
* Single Post Page
* Toasts And Loaders
* Responsive

# Tech Stack
* React JS
* React Router v6
* React Hooks like-
  * useContext()
  * useReducer()
  * useState()
  * useParams()
  * useNavigate()
* React Toastify for Toasts
* Mock Backend with MockBee https://mockbee.netlify.app/
* Vanilla CSS 
* Font Awesome Icons

# How to run locally
```
$ git clone https://github.com/amulyaparab/meowments.git
$ cd meowments
$ npm install
$ npm start 
```
